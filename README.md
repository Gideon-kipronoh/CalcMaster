# CalcMaster

Calcmaster is a basic graphical user interface (GUI) calculator built using java that allows users to perform simple arithmetic calculations. It's designed for simplicity and ease of use.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Clearing the input
